# Clothes Donation Website

This is a web-based clothes donation system developed using HTML, CSS, JavaScript, PHP, and MySQL.

## Features
- User Registration & Login
- Donate Clothes Form
- Pickup Address Form
- Database Connectivity
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## Future Improvements
- Admin Dashboard
- Email Notifications
- Tracking System
